Belajar HTML #16: Hosting di Netlify

Langkah-langkah:

  1. Buat akun di Netlify.com (bisa dengan GitHub).
  2. Push semua file HTML ke repository GitHub.
  3. Di Netlify, klik "Add new site" > "Import an existing project".
  4. Pilih GitHub, authorize, pilih repository.
  5. Biarkan pengaturan default, klik "Deploy site".
  6. Situs akan live dengan URL otomatis (dapat diubah).

Contoh file netlify.toml (opsional):

[build]
publish = "."

Setiap push ke GitHub, Netlify akan otomatis redeploy.
